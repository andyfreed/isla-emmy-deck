# Dev Roadmap & Order-of-Operations (the discipline)

This is the agreed development order. Claude acts as the guardrail: if work drifts
ahead of the current phase, flag it and steer back to the critical path. The rule is
about the CRITICAL PATH (the make-or-break work), not about banning fun.

## The order (phases — don't skip ahead)
1. **Concept** — the one-line hook. ✅ DONE: "Two sisters dance-battle unlucky
   zodiac creatures back to the Moon."
2. **Pre-production — FIND THE FUN** ← WE ARE HERE.
   Prove the core loop is fun with PLACEHOLDER art. Lock art *direction* (done),
   not final assets.
3. **Production** — build a vertical slice, then breadth; art + music in parallel.
4. **Polish** — final music/SFX, juice, balance, ship.

## Current phase: Pre-production — find the fun
**Goal:** make the core loop genuinely fun with placeholders:
  village → balloon to an island → explore → find a grumpy zodiac creature →
  Funk-Off battle (rhythm + zodiac clash, losable) → calm it → it returns to the
  Moon + grants its power → back to village.

**Focus now:** the **Funk-Off battle** (the core verb). Make ONE battle feel great.

**Gate to advance to Production:** Emmy plays the loop and *wants another go*. (She's
the target player — her reaction is the design data.)

## Parked until the gate passes (don't let these block the loop)
- **iPad/iOS port** (user has Apple dev account + shipped iOS before; Godot exports
  iOS; TestFlight = best auto-update). Needs: virtual-joystick touch controls +
  local Xcode builds. Do AFTER the game is good — it's distribution, not game.
- Mass-producing environment art (home-island ground/props can stay placeholder).
- Final music & SFX (temp/placeholder only for now).
- Building all 12 creatures / all islands (breadth before the loop is fun = the #1
  indie trap).
- Polish/juice passes on unfinished systems.

## Always OK (not on the critical path, but good)
- The kids generating art / dreaming up lore — it locks art direction and keeps
  Emmy & Isla invested. Just don't let it *replace* proving the loop.
- The auto-update pipeline / infrastructure.

## Definition of the vertical slice (the Production target)
ONE island, fully playable end-to-end with the complete loop above, with real art
for THAT slice only. If the slice is fun and looks right, replicate it across the 12.
